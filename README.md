# Sign-Language-Alphabet-Classifier
This project is designed to recognize sign language alphabet letters from hand gesture images using deep learning. The dataset is organized into labeled folders, where each folder represents a specific letter or class.

## Features
- Recognizes hand sign images as alphabet classes
- Uses deep learning for automatic feature extraction
- Preprocesses images by resizing, RGB conversion, and normalization
- Classifies images into 36 different classes
- Built with the Keras Functional API
